<h1>Consumo de Almoço</h1>

<p>O seguinte projeto foi desenvolvido como prática da Lingugem C no curso de Análise e Desenvolvimento de Sistemas.</p>

<p>Por meio dele, foi desenvolvidas habilidades como lógica de programação, resolução de problemas cotidianos e uso de vetores na linguagem  de programação C</p>

<h2>Objetivo</h2>
<p>O objetivo de tal projeto é permitir otimizar o uso de dados que são processados diariamente. Com o uso de vetores é possível adicionar os valores sobre o consumo de almoço diário e no final obeter a média do consumo semanal. Além disso, é possível costumizá-lo para o uso em outros contextos permitindo um uso mais eficiente.</p>
